组播地址生成器

此工具可将多个IP地址和端口与组播ID拼接，生成完整的组播地址。只需在左侧输入框中每行输入一个IP:端口，选择协议类型，输入组播ID，然后点击转换按钮即可。

输入设置
行数: 0

示例格式

输入IP:端口: 125.66.145.243:8848

组播ID: 239.94.2.51:5140

选择协议: UDP

结果: http://125.66.145.243:8848/udp/239.94.2.51:5140

转换结果
生成数量: 0

使用说明

1. 在左侧输入IP和端口，每行一个

2. 选择组播协议 (UDP/RTP/RTSP/HTTP)

3. 输入组播ID (格式: 组播IP:端口)

4. 点击"转换地址"按钮生成结果

5. 使用"复制"或"下载结果"保存生成内容
