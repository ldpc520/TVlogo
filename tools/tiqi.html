🔍 网页内容提取工具 - 无CORS限制版

无需担心CORS安全限制，多种方式提取网页内容

📋 方法一：手动粘贴网页内容

这是最简单且完全无CORS限制的方法。您只需要从浏览器手动复制网页内容，然后粘贴到这里进行提取。

操作步骤：

1
打开您要提取内容的网页（如您之前提到的那个URL）
2
按 Ctrl+A (Windows) 或 Cmd+A (Mac) 全选页面内容
3
按 Ctrl+C (Windows) 或 Cmd+C (Mac) 复制内容
4
将内容粘贴到下面的文本框中

🔧 方法二：使用浏览器扩展绕过CORS

安装浏览器扩展程序，一劳永逸地解决CORS问题。这需要您安装一个浏览器扩展，但之后可以自动处理CORS限制。

💡 推荐扩展：

以下是各浏览器可用的CORS扩展，安装后启用即可：

🌐 Chrome/Edge浏览器：

  • Allow CORS: Access-Control-Allow-Origin - 最受欢迎的CORS扩展
  • Moesif CORS - 轻量级，简单易用
  • CORS Unblock - 功能全面，支持自定义

🦊 Firefox浏览器：

  • CORS Everywhere - Firefox专用，效果很好
  • Allow CORS Access-Control-Allow-Origin - 也有Firefox版本
安装CORS扩展后，点击按钮即可直接获取网页内容

⚙️ 方法三：使用本地代理工具

这是最专业的方法，通过在本地运行一个简单的代理服务器来绕过CORS限制。适合有一定技术基础的用户。

📁 最简单的本地代理方案：

创建一个名为 proxy.html 的文件，内容如下：

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>简易代理</title>
</head>
<body>
<script>
// 从URL参数获取目标地址
const urlParams = new URLSearchParams(window.location.search);
const targetUrl = urlParams.get('url');

if (targetUrl) {
fetch(targetUrl)
.then(response => response.text())
.then(data => {
document.body.innerHTML = '<pre>' + data + '</pre>';
})
.catch(error => {
document.body.innerHTML = '错误: ' + error;
});
}
</script>
</body>
</html>

使用方法：

  1. 将上面的代码保存为 proxy.html
  2. 用浏览器打开 file:///路径/proxy.html?url=您要获取的网址
  3. 页面会显示网页内容，复制内容到方法一中使用
⚠️ 注意： 这种方法需要以 file:// 协议打开本地HTML文件，部分浏览器可能有安全限制。
